# FAIR theory: The Allostatic Theory of Oxytocin 

# Description

This is a FAIR implementation of Quintana and Guastella's Allostatic Theory of Oxytocin, which describes the role of oxytocin in human behavior


# Interoperability

The theory is implemented in the DOT language for describing graphs.

### Rendering the theory using graphviz

See the graphviz manual for more information: https://graphviz.org
